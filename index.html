實作心得與反思

在學習的過程中，我一步步跟著影片中的老師操作，剛開始常常會遇到不懂的地方，很多語法和標籤都需要先記住用法，才能順利完成練習。 不過我發現，許多 HTML 標籤其實都有類似的邏輯，只要多做幾次，就會慢慢上手，也會對它們的功能和使用方式越來越熟悉。 至於 CSS，一開始不太會用時，我會上網查資料或詢問 ChatGPT，遇到問題就嘗試解決。透過這樣不斷練習的過程，雖然還沒辦法自己寫出完整的網頁， 但至少現在我已經能夠看懂網頁架構，對 HTML 和 CSS 的語法也有基本的判讀能力。
